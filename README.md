# Homebrew Tap for lix-agents

## Installation

```bash
brew install lix-it/lix-agents/lix-agents
```

Or add the tap first:

```bash
brew tap lix-it/lix-agents
brew install lix-agents
```

## About

This tap contains the Homebrew formula for [lix-agents](https://github.com/lix-it/lix-agents), a CLI tool for obtaining temporary Lix API tokens with email-based approval.
